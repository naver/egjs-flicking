# flickingReactiveAPIAdapter

> Internal reactive API adapter for Flicking that manages state and event listeners

## Description

This adapter is used internally by framework-specific packages (react-flicking, vue-flicking, etc.) to provide reactive API support. Users rarely need to use this directly.

## Type

```typescript
ReactiveSetupAdapter<FlickingReactiveObject, FlickingReactiveState, "moveTo", FlickingReactiveData>
```
