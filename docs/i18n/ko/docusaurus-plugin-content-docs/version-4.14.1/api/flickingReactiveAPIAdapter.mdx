Flicking의 상태와 이벤트 리스너를 관리하는 내부 반응형 API 어댑터<br />이 어댑터는 react-flicking, vue-flicking 등의 프레임워크별 패키지에서 내부적으로 사용되어<br />반응형 API 지원을 제공합니다. 사용자가 직접 사용할 일은 거의 없습니다.

**Returns**: 
- Flicking 상태와 메서드를 포함한 반응형 객체

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|onInit||||반응형 객체가 초기화될 때 호출되는 콜백|
|onDestroy||||반응형 객체가 파괴될 때 호출되는 콜백|
|setMethods||||사용 가능한 메서드를 설정하는 함수|
