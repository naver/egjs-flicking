### adaptive
Update height of the viewport element after movement same to the height of the panel below.
This can be only enabled when `horizontal=true`

<Columns>
  <ColumnItem is={6}>

  ```js
  adaptive: false
  ```

  </ColumnItem>
  <ColumnItem is={6}>
    <Flicking adaptive={false}>
      <div className="flicking-panel has-background-primary has-text-white is-size-1">1</div>
      <div className="flicking-panel high has-background-primary has-text-white is-size-1">2</div>
      <div className="flicking-panel higher has-background-primary has-text-white is-size-1">3</div>
    </Flicking>
  </ColumnItem>
</Columns>
<Columns>
  <ColumnItem is={6}>

  ```js
  adaptive: true
  ```

  ```css
  .flicking-viewport {
    transition: height 500ms;
  }
  ```

  </ColumnItem>
  <ColumnItem is={6} className="has-text-centered">
    <Flicking adaptive={true}>
      <div className="flicking-panel has-background-primary has-text-white is-size-1">1</div>
      <div className="flicking-panel high has-background-primary has-text-white is-size-1">2</div>
      <div className="flicking-panel higher has-background-primary has-text-white is-size-1">3</div>
    </Flicking>
  </ColumnItem>
</Columns>

:::info
You should add CSS `transition` property to `flicking-viewport` to animate height.
:::
