# Reactive API
`Reactive API`는 옵션과 이벤트가 아닌 상태로써 직관적이고 변화에 있어 효율적인 API를 말합니다.
(상태 부가 설명)

또한 `React`의 경우에도 다름 프레임워크와 유사하게 사용하는 속성의 변화에만 업데이트하는 기능을 지원합니다.


* 사용법 간소화
* 외부 컴포넌트와의 연계
* 커스텀에 대한 기대


현재 JavaScript, React, Vue 2, 3 (Composition API)를 지원하고 있으며 다른 프레임워크도 확장할 예정입니다.
코드

<Tabs
  defaultValue="js"
  lazy={true}
  values={[
    { label: "JavaScript", value: "js" },
    { label: "React", value: "react" },

  ]}>
  <TabItem value="js">

```js
import Flicking, { connectFlickingAPI } from "@egjs/flicking";

const flicking = new Flicking("#el");
const state = connectFlickingAPI(flicking);

state.subscribe("currentIndex", nextValue => {
  console.log("currentIndex", nextValue);
});
```

  </TabItem>
  <TabItem value="react">

```jsx
import Flicking from "@egjs/react-flicking";
import { AutoPlay } from "@egjs/flicking-plugins";

export default () => {
  const plugins = [new AutoPlay({ duration: 2000, direction: "NEXT", stopOnHover: false })];

  return (<Flicking plugins={plugins}>
    { ... }
  </Flicking>);
};
```

  </TabItem>
  <TabItem value="vue">

```html title="template"
<flicking :plugins="plugins">
  {{ ... }}
</flicking>
```

```js title="script"
import { AutoPlay } from "@egjs/flicking-plugins";

const plugins = [new AutoPlay({ duration: 2000, direction: "NEXT", stopOnHover: false })];

module.exports = {
  data: function () {
    return {
      plugins
    }
  }
}
```
