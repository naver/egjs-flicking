## Variable Size Panels
Flicking uses panel element's real size, and doesn't modify them.
<Flicking className="my-4" circular={true}>
  <div className="flicking-panel has-text-white is-size-1 has-background-primary" style={{ width: "120px" }}>1</div>
  <div className="flicking-panel has-text-white is-size-1 has-background-danger" style={{ width: "20%" }}>2</div>
  <div className="flicking-panel has-text-white is-size-1 has-background-info" style={{ width: "500px" }}>3</div>
  <div className="flicking-panel has-text-white is-size-1 has-background-warning" style={{ width: "300px" }}>4</div>
  <div className="flicking-panel has-text-white is-size-1 has-background-success" style={{ width: "100%" }}>5</div>
</Flicking>
