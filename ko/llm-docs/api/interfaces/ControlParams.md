# ControlParams

> A current parameters of the Camera for updating `AxesController`

## Properties

### circular

**Type:** `boolean`

A Boolean indicating whether the [circular](https://naver.github.io/egjs-flicking/llm-docs/api/classes/Flicking.md#circular) option is enabled

### position

**Type:** `number`

Current camera position

### range

**Type:** `{
        min: number;
        max: number;
    }`

A moveable range for Camera
