# ReachEdgeEvent

> Event that fires when camera reaches the maximum/minimum range

## Properties

### direction

**Type:** `ValueOf<typeof DIRECTION>`

Direction indicates whether the camera's position is at the maximum range([DIRECTION.NEXT](https://naver.github.io/egjs-flicking/llm-docs/api/variables/DIRECTION.md)) or minimum range([DIRECTION.PREV](https://naver.github.io/egjs-flicking/llm-docs/api/variables/DIRECTION.md))
