# WillRestoreEvent

> Event fires BEFORE returning to the current panel when drag doesn't reach threshold.

## Description

Fired when user drag amount does not reach [threshold](https://naver.github.io/egjs-flicking/llm-docs/api/classes/Flicking.md#threshold) and the camera starts returning to [currentPanel](https://naver.github.io/egjs-flicking/llm-docs/api/classes/Flicking.md#currentpanel).

## Properties

### direction

**Type:** `ValueOf<typeof DIRECTION>`

Moving direction relative to previous position of the camera

### index

**Type:** `number`

Index of the panel to restore

### isTrusted

**Type:** `boolean`

Boolean that indicates whether the event was generated by a user action

### panel

**Type:** `Panel`

Panel to restore
