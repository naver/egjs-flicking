# RestoredEvent

> Event that fires AFTER Flicking has returned to [currentPanel](https://naver.github.io/egjs-flicking/llm-docs/api/classes/Flicking.md#currentpanel).

## Description

Fired after the restoration animation completes and the camera has returned to the current panel.

## Properties

### isTrusted

**Type:** `boolean`

Boolean that indicates whether the event was generated by a user action
