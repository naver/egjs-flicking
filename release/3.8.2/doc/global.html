EventDefinition

Types that can be used when attaching new event definition as generic on a class

클래스 타입 등록시 사용가능한 타입

Example
new SomeClass<{
  // Using it as object
  evt0: {
    param0: number;
    param1: string
  };
  // Using it as function with other arguments
  evt1: (arg0: {
    param0: number;
    param1: string
  }, arg1: string, arg2: boolean) => boolean;
}>
